# BNotee

一个运行在浏览器上的笔记程序。 [GitHubPages](https://bhznjns.github.io/BNotee/dist/)

* * *

## 行内样式对照 Inline Styles

| 符号 Marks | 样式 Styles |
|  ----  |  ---- |
| ##Bold## | **##Bold##** |
| \``Code\`` | <code>\``Code\``</code> |
| \__Underlined\__ | <u>\__Underlined\__</u>
| //Italic// | <i>//Italic//</i> |
| --Deleted-- | <del>--Deleted--</del> |
| ::Weaken:: | <span style="opacity: .75">Weaken</span> |

* * *

## 已实现 Finished

- 标题(Headings (1 ~ 4))
- 段落(Paragraph)
- 分割线(Horizonal Rule)
- 列表块(Ordered List && Unordered List)
- 表格块(Table)
- 详情块(Details)
- 黑暗模式(Dark Mode)
- 触屏模式(Touch Mode)
- 全屏(阅读模式 Read Mode)
- 行内样式(Inline Style)
- 导入 / 导出(Import / Export)
- 代码块(Code Block)
- .md 导出(Markdown exports)
